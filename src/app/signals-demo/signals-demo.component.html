Writable Signal: Counter

This example demonstrates a writable signal called counter. It is initialized to 0 and can be updated directly.

Current counter value: {{ counter() }}

Computed Signal: Double Counter

This example demonstrates a computed signal called doubleCounter. Its value is derived from the counter signal and automatically updates when counter changes. It always displays the double of the counter's current value.

Current doubleCounter value: {{ doubleCounter() }}

Effect: Log Counter Changes

This example demonstrates an effect that listens to changes in the counter signal. Whenever the counter value changes, a message is logged to the browser's console. Open your browser's developer console to see the logs.
