Formly Integration Examples

This section will demonstrate how to use ngx-formly for dynamic forms.

Setting up ngx-formly

To use ngx-formly, you need to install it and its peer dependencies. For a typical Angular project with Material or Bootstrap, you might run:


ng add @ngx-formly/schematics --ui-theme=material # Or bootstrap, ionic, kendo, etc.

Or manually install:


npm install @ngx-formly/core @ngx-formly/material # Replace material with your chosen UI theme
# You also need Angular Material/CDK if using the material theme
npm install @angular/material @angular/cdk

Then, you need to import `FormlyModule` and the UI theme module (e.g., `FormlyMaterialModule`) into your application module or the specific standalone component where you intend to use Formly.

For a standalone component like this one, you would import them directly:


import {{ '{' }} FormlyModule } from '@ngx-formly/core';
import {{ '{' }} FormlyMaterialModule } from '@ngx-formly/material'; // Or your chosen theme
import {{ '{' }} ReactiveFormsModule } from '@angular/forms'; // Formly relies on ReactiveFormsModule

@Component({{ '{' }}
  selector: 'app-formly-wiki',
  standalone: true,
  imports: [
    CommonModule,
    ReactiveFormsModule, // Required for Formly
    FormlyModule, // For Formly core
    FormlyMaterialModule // Or your chosen UI theme module
    // ... other necessary Angular Material modules if using Material theme (e.g., MatInputModule, MatButtonModule)
  ],
  templateUrl: './formly-wiki.component.html',
  // ...
})
export class FormlyWikiComponent {{ '{' }}
  // ...
}

Note: This component will attempt to use Formly with Material. Ensure necessary dependencies are conceptually 'installed' for the examples to work.
